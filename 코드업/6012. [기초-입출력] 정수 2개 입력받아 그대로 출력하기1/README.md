## 문제 설명

본 문제는 python 의 빠른 기초 학습을 위해 설계된 문제로서 python 코드 제출을 기준으로 설명되어 있습니다. 
------

줄을 바꿔 정수(integer) 2개를 입력받아 줄을 바꿔 출력해보자.

예시
a = input() 
b = input()
a=int(a)
b=int(b)
print(a)
print(b)
과 같은 방법으로 가능하다.

## 입력

1
2

## 출력

1
2
