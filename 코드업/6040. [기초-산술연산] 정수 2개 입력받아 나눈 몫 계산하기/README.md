메모리:  KB, 시간:  ms

## 분류

## 문제 설명

정수 2개(a, b) 를 입력받아 a를 b로 나눈 몫을 출력해보자.

예시
...
print(a//b)

참고
python언어에서는 나눈 몫을 계산하는 연산자(//, floor division)를 제공한다.
a//b 와 같이 작성하면, a를 b로 나눈 몫(quotient)을 계산해준다.
프로그래밍언어에 따라 이렇게 몫을 계산해주는 연산자가 없는 경우도 있다.

실수로 나눈 몫이 어떻게 계산될지도 생각해보자.

## 입력

```
10 3
```

## 출력

```
3
```
