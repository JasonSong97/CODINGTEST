## 문제 설명

실수 2개(f1, f2)를 입력받아 곱을 출력하는 프로그램을 작성해보자.

예시
...
m = f1 * f2
print(m)

참고
수 * 수는 곱(multiplication)이 계산된다.

## 입력

```
0.5 2.0
```

## 출력

```
1.0
```
